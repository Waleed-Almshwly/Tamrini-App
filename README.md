# 🚀 Getting Started

## Requirements

- Android Studio
- Android SDK
- Kotlin Support
- Gradle

---

# ▶️ Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/Tamrini-App.git
```

1. Open the project in Android Studio  
2. Wait for Gradle Sync  
3. Run the app on Emulator or Physical Device
